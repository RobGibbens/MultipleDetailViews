Multipledetailviews

Download this project as a .zip file Download this project as a tar.gz file

A MonoTouch port of the MultipleDetailViews iPad sample app from Apple. "This sample shows how you can use UISplitViewController to manage multiple detail views. The application uses a split view controller with a table view controller as the root view controller. When you make a selection in the table view, a new view controller is created and set as the split view controller's second view controller.
